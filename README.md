# Weather App

A simple and responsive weather application built with HTML, CSS, and JavaScript that fetches real-time weather data using the [WeatherAPI](https://www.weatherapi.com/).

## Features

- Search weather by city name
- Displays current temperature (°C)
- Shows humidity and wind speed
- Dynamic weather icon based on weather condition
- Clean and responsive UI

## Tech Stack

- **HTML5** — Page structure
- **CSS3** — Styling and responsive layout
- **JavaScript (ES6+)** — Fetching API data and updating the DOM
- **WeatherAPI** — Live weather data

## Project Structure


## How to Use

1. Clone or download the project.
2. Open `index.html` in any web browser.
3. Type a city name in the search box and press **Search** o
4. The app will display the current temperature, humidity, wind speed, and a weather icon.

## API Setup

This project uses [WeatherAPI.com](https://www.weatherapi.com/).

1. Sign up at [weatherapi.com](https://www.weatherapi.com/) to get a free API key.
